# 阶段6：初始化

> 将所有设计整合，生成青色协作空间。

---

## 目标

- 整合前面所有阶段的设计
- 生成完整的青色协作配置
- 初始化记忆结构
- 确保青色协作空间自动生效

---

## 生成文件清单

初始化将生成以下文件：

```
~/.workbuddy/teal-collab/
├── SOUL.md                  # 使命文档
├── IDENTITY.md              # 身份定义
├── BOUNDARIES.md            # 边界规则
├── COMMITMENTS.md            # 承诺框架
├── ROLES/                   # 角色定义目录
│   ├── role-1.md
│   ├── role-2.md
│   └── ...
├── memory/                   # 记忆文件目录
│   └── YYYY-MM-DD-init.md   # 初始化记录
└── prompts/                  # 自动加载片段
    ├── system-prompt.md      # 系统提示词片段
    └── reminder-prompt.md    # 定期提醒片段
```

## 初始化检查清单

生成文件后，确认：

- [ ] 所有文件已生成到正确位置
- [ ] 文件内容与设计阶段确认的一致
- [ ] 用户已阅读并认可核心文件
- [ ] 用户知道如何开始第一次青色协作
- [ ] 用户知道如何发起复盘或调整
